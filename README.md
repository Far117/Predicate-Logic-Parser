# Predicate Logic Parser
Parses arbitrary predicate logic statements, and can pretty-print a truth table for all variables.

Available operators:

| Name          |Operator |
| -----         | ----    |
| NOT           | `!`     |
| AND           | `&&`    |
| NAND          | `&`     |
| OR            | `\|\|`  |
| XOR           | `\|`    |
| Implication   | `->`    |
| Bidirectional | `<->`   |
